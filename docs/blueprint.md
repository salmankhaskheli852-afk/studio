# **App Name**: InvestPro Wallet

## Core Features:

- Investment Options: Display three investment options with details.
- Wallet Functionality: Enable deposit and withdrawal through JazzCash and Easypaisa. User enters account holder name, number, and amount.
- Deposit Verification: Require users to input the account holder's name, number, transaction ID (TID), and a screenshot for deposit verification.  Also show deposit methods.
- Admin Wallet Details: Display admin wallet details for deposits (number, name, and wallet name).
- Navigation: Implement navigation with four options: Home, Wallet, Invest, and Invite.
- Invite and Referrals: Provide an invitation link and display the user's referral code.
- Transaction History: Display user transaction history (deposits and withdrawals) in the Wallet section.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), suggestive of clear skies and financial clarity. The blue will promote feelings of trust.
- Background color: Very light cyan (#E0FFFF), which harmonizes with the deep sky blue to keep the theme airy and light.
- Accent color: Lime green (#32CD32) to represent the rewards from investment.
- Body and headline font: 'PT Sans', sans-serif.
- Use financial and money-themed icons.
- Keep the design clean and modern, focusing on usability.
- Use subtle animations to enhance the user experience.
